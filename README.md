# Holmstad-Library-Kiosk
Library catalog for Holmstad residents
Download and install Autohokey version 1 https://www.autohotkey.com/
Download and install Chrome Browser.  Other browsers maybe used, but changes may be required in library.ahk.
Add kiosk.bat to start at logon.  https://benjamingift.wordpress.com/2012/03/05/starting-an-autohotkey-script-at-startup-with-windows-7/
