Holmstad Library Kiosk Instructions

This tool allows one to search the Holmstad library by various methods.

Power Button

The power button is on the left side. If the button is yellow the power is off. If it is on then the power button is blue and a mouse movement or striking a key on the keyboard should wake the PC.

* The PC is slow so patience is required to wait for the PC to complete powering up.

Logging On

After The PC is powered on you will be shown a logon screen. Select the icon labeled Kiosk.

Kiosk Screen

You will be presented with the "LibraryThing" web site. The Home button will take you back to this screen.

Clicking on "File" button will give you options to return to this screen, Exit or Power off.

The "?" buton will present this help.

Other options are explained below.

Search Overview

Type your search word in the search box (labeled "Search this library") to find books quickly.

Sorting

Click on any column heading to sort on that column. Clicking again on the column will reverse the sort.

  • Clicking on the column title will give information about that item
  • Clicking on an authors name will give information about the author.
  • Clicking on a tag from the tag field will show all books referenced by the tag.

Collections

A Search can be made on all books or categorized by "Fiction" or "non-Fiction". Use the All collections pull down to search the entire collection.

Search Overview

  • Use the search box (labeled "Search your library") to find books quickly. Use the dropdown to limit your search to specific fields.
  • The "Advanced search" and "Search tips options will give instruction on how to do better searches.
