Holmstad Library Kiosk Instructions

This tool allows one to search the Holmstad library by various methods.

Power Button

The power button is on the left side. If the button is yellow the power is off. If it is on, then the power button is blue and a mouse movement or striking a key on the keyboard should wake up the PC.

* The PC is slow so patience is required to wait for the PC to complete powering up.

Logging On

After The PC is powered on you will be shown a logon screen. Select the icon labeled Kiosk.

Kiosk Screen

You will be presented with the "LibraryThing" web site. The Home button will take you back to this screen at any time.

  • Clicking on "File" button will give you options to return to this screen, Exit or Power off.

  • Holding down he alt key and pressing the left arrow button take you to the previous screen.

    The "?" button will present this help.

    Other options are explained below.

Search Overview

Type your search word(s), (for example author name or title) in the search box (labeled "Search this library") on the top right of the screen. Then use the mouse to click on the word "Search". Our books written by that author will will be listed, or if we have that title it will be listed.

Sorting

Click on any column heading to sort on that column. Clicking again on the column will reverse the sort.

  • Clicking on any item in a column will give information about that item
  • Clicking on an authors name will give information about the author.
  • Clicking on a book title will give more information about the book.

Collections

A Search can be made on all books or categorized by "Fiction" or "non-Fiction". Use the All collections pull down to search the entire collection.

Search Overview

  • Use the search box (labeled "Search your library") to find books quickly. Use the dropdown to limit your search to specific fields.
  • The "Advanced search" and "Search tips options will give instruction on how to do better searches.
