Holmstad Library Kiosk Instructions

This tool allows one to search the Holmstad library by various methods.

Power Button

The Power Button is on the left side.

If it is not lit the computer is powered off.

If the button is amber the computer is in a standby or sleep state. Moving the mouse or pushing a keyboard button should wake up the PC.

If the button is blue The computer is powered on and functioning normally.

* The PC is slow so patience is required to wait for the PC to complete powering up.

Logging On

After The PC is powered on you will be shown a logon screen. Select the icon labeled Kiosk.

Kiosk Screen

You will be presented with the "LibraryThing" web site. The Home button (1) will take you back to this screen at any time.

  • Clicking on "File" button (1) will give you options to return to this screen, Exit or Power off.

  • Holding down the alt key and pressing the left arrow button take you to the previous screen.

    The "?" button (2) will present this help document.

    Other options are explained below.

Search Overview

Type your search word(s), (for example author name or title) in the search box (4) (labeled "Search this library") on the top right of the screen. Then use the mouse to click on the word "Search" (4). Our books written by that author will will be listed, or if we have that title it will be listed.

Sorting

Click on any column heading (6) to sort on that column. Clicking again on the column will reverse the sort.

Clicking on any item (7) in a column will give more information about that item.

  • Clicking on an authors name will give information about the author.
  • Clicking on a book title will give more information about the book.

Collections

A Search can be made of all books, only "Fiction" or only "non-Fiction". Use the All collections pull down (3) to search the entire collection.

Search Overview

  • Use the search box (4) (labeled "Search this library") to find books quickly. Use the search dropdown (5) to limit your search to specific fields.
  • The "Advanced search" and "Search tips options will give instruction on how to do more in-depth searches.
